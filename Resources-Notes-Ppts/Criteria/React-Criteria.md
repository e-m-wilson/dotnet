# React

| Criteria Difficulty | Exit Criteria                                                                                                  |
| ------------------- | -------------------------------------------------------------------------------------------------------------- |
| Required            | Describe the use of functional components                                                                      |
| Required            | Describe how Single Page Applications work differently from Multi Page Applications.                           |
| Required            | Utilize and explain the most common hooks: useState, useEffect, etc.                                           |
| Required            | Pass props into a Component.                                                                                   |
| Required            | Explain the use of JSX in a React application.                                                                 |
| Preferred           | Use Axios or Fetch to make HTTP requests.                                                                      |
| Preferred           | Route users between Components through the use of BrowserRouter.                                               |
| Preferred           | Leverage route guards to change the routing behavior based on the given state                                  |
| Preferred           | Describe React paradigms.                                                                                      |
| Preferred           | Implement Function Components.                                                                                 |
| Preferred           | Leverage NPM libraries in a React Project to add functionality.                                                |
| Stretch             | Use createContext() to make multiple contexts.                                                                 |
| Stretch             | Use Context.Provider tags to wrap components and distribute application state.                                 |
| Stretch             | Use advanced hooks to manipulate context in applications.                                                      |
| Stretch             | Use Jest and a React testing library to test components.                                                       |
| Stretch             | Leverage advanced routing techniques to create parent-child routing, or through passing variables into routes. |
